Using this package you can easily display maps on your website.

Supported map services: 
 * Google Maps
 * OpenStreetMap
 * Bing Maps
 * MapQuest
 * Yandex Maps
 * MapKit (beta)
 
 
 Source code, full documentation & issues: https://gitlab.com/gonoware/laravel-maps
